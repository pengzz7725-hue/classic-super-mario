# 🍄 经典超级马里奥 (Classic Super Mario)

一个用纯 HTML + Canvas + 原生 JavaScript 实现的经典横版超级马里奥小游戏，**零依赖、单文件**，双击即玩。

## 🎮 在线体验

直接用浏览器打开 [`index.html`](./index.html) 即可游玩，无需安装任何东西。

> 如果开启了 GitHub Pages，也可以在线访问：`https://<你的用户名>.github.io/<仓库名>/`

## 🕹️ 操作说明

| 按键 | 功能 |
| --- | --- |
| `←` / `→`（或 `A` / `D`） | 左右移动 |
| `空格` / `↑` / `W` | 跳跃（长按跳得更高） |
| `回车` / `空格` | 开始 / 重新开始游戏 |

## ✨ 游戏特性

- 🏃 经典物理手感：加速度、惯性、摩擦力、可变高度跳跃
- 👾 敌人 Goomba：踩头消灭，撞到则受伤掉血
- 🪙 金币与问号块：顶砖出金币，集满 100 枚加一条命
- 🧱 可破坏砖块、绿色管道、滚动镜头
- 🚩 旗杆终点 + 城堡通关动画
- ❤️ 分数、金币、计时、生命值 HUD

## 🛠️ 技术栈

- 原生 `<canvas>` 2D 渲染
- 纯 JavaScript（无任何框架/库）
- 单 HTML 文件，约 600 行

## 📦 本地运行

```bash
# 克隆仓库
git clone https://github.com/<你的用户名>/<仓库名>.git
cd <仓库名>

# 直接用浏览器打开 index.html，或起一个本地服务器：
python -m http.server 8000
# 然后访问 http://localhost:8000
```
